VDN-Theme

This wordpress theme is used in VDN website. 
This is a proprietary theme, aimed to the specific use of VDN website.

This theme overrides templates for following elements :
- Zerif-Lite theme : https://themeisle.com/themes/zerif-lite/
- Ultimate Members plugin : https://ultimatemember.com/
- Modern Tribe Calendar : https://theeventscalendar.com/


This plugin is meant to be used in association with VDN-Companion :
https://github.com/philsabaty/vdn-companion
